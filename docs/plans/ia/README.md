# IA Shards

Information Architecture shards created by `/decompose-architecture` and filled by `/write-architecture-spec`.

## Naming Convention

Shards are numbered sequentially: `00-*.md`, `01-*.md`, ..., `NN-*.md`. An `index.md` file maps shard numbers to domain boundaries.

## Completion Requirement

All IA shards must reach **Complete** status before `/write-be-spec` begins. Run `/audit-ambiguity ia` to verify.

## Subdirectories

| Directory | Description |
|---|---|
| `deep-dives/` | Deep dive documents for features too complex for the main shard |
