# Raw Memory Layer

This folder is the append-only capture layer of the shared memory vault.

## Folders
- `daily/` — human-readable daily notes
- `events/` — raw event captures
- `sessions/` — raw session streams
- `assets/` — attachments and supporting files

## Rules
- Do not treat this as curated memory
- Let compile jobs promote useful material into `wiki/`
- Prefer writing through MCP tools or runtime hooks
